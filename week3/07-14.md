# be-DailyJob 

> 이 회고는 작업량을 보고하기 위한 문서가 아니라, 
> 오늘 내가 어떤 판단을 했고 어떤 개발 행동을 반복하거나 바꿔야 하는지 확인하기 위한 기록입니다. 
> 각 항목은 길게 쓰기보다, 사실/판단/근거/다음 행동이 드러나게 구체적으로 작성하세요.

### 1. 오늘 완료한 것
- 오늘 실제로 끝낸 기능/수정/학습: 게시판 상세 마이너한 요구사항들 구현, PR 메세지에 나타난 에러들 해결, 파일 업로드 버튼 및 파싱 로직 및 클래스 추가
- (오늘의 목표 중에서) 아직 끝내지 못한 부분: 실제 서버 파일 업로드 시 File로 저장하고 imageUrl을 DB에 저장하는 부분, 자바스크립트 파일 업로드 관련 로직

### 2. 오늘 아쉬웠던 행동(개발과 관련된 것)
- 비효율적이었던 접근/놓치거나 부족했던 점: 어느 정도 예상됐던 에러나 예외 상황에 대해서 처리를 미뤄뒀던 점.
- 그렇게 된 원인: 구조가 잡히고 나중에 한번 모아서 처리하는 게 편할 것이라고 생각함.
- 다음에는 다르게 해볼 구체적인 행동: 예상되는 예외 상황에 대해서 될 수 있으면 바로 처리하는 것이 최선이겠지만 그것이 안되면 주석으로 Todo라도 꼼꼼히 남겨두기.
